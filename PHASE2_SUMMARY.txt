FIELD ADDITIONS TO PROPERTY MODEL
================================================================================

The following six fields must be added to the Prisma Property model:

  zoningCode          String?   # e.g., "R1", "C2", "INDUSTRIAL"
  zoningDescription   String?   # e.g., "Single-Family Residential"
  schoolDistrict      String?   # e.g., "Greenville County School District 50"
  waterService        String?   # e.g., "City of Greenville Water Authority"
  sewerService        String?   # e.g., "City of Greenville Sewer System"
  fireDistrict        String?   # e.g., "Greenville Fire District 1"

All fields are optional (String?) to accommodate properties with incomplete data.

Migration:
  npx prisma migrate dev --name add_enhanced_gis_fields

This will generate a migration file and apply the schema changes to the database.
